### 🎮 How to use?

#### Cargo run!
```
$ cargo run --release
```

Commands 

```md
Move: ↑,↓,←,→ or w,s,a,d
Quit : q
New game : r
Pause : p
```

--------------

#### Installation

For rust users

```
$ cargo install rust-15-puzzle-cli
```
